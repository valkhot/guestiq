The design calls here:

1 · Core is complete, full stop. The screen says it plainly — "a real, complete contribution… it counts" — so finishing at core never feels like the lite version. This is the canon guardrail: breadth (core) is what clears the convergence floor, so it must be honoured.
2 · The deeper option is a compliment, not a nag. "You clearly know this guest" — it's offered once, mid-read, framed as recognition of expertise, then never again.
3 · Neither choice reads as quitting. Both labels point forward — "A few more →" and "See your read →" — so finishing at core feels like moving on to the payoff, not stopping short. The affirmation ("a real, complete contribution… it counts") sits right above. Each label is plain enough for a non-native speaker to read at a glance.
4 · "Results" is deliberately avoided. The end-of-read reflects the agent's expertise back — it is not a score or grade. "Results-screen" was retired in the pivot for exactly this reason, so the button promises a read to see, never a mark.

Content-requirement check: Now governed by NFR-TONE-01 (professional / expert-witness register) and NFR-TONE-02 (plain-language / ESL — common short words, no idioms in controls). The idiomatic "Go deeper" has been replaced with the plainer "A few more →" under that rule.
